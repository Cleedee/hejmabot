# Hejmabot

## Comandos

start - Apresentação do bot
estoque - Lista dos itens em casa
status - Itens perto do vencimento
usar - Consumir os itens
sugerir_jantar - Sugirir uma receita
lista_compras - Monta uma lista de compras com os itens que acabaram
pergunta - Para perguntas sobre o banco de dados
